# AI-BOS Quick Start Guide

**Welcome to AI-BOS!** 
If you have 2 minutes, here is exactly how to set up your business data to see the AI CEO in action.

## 1. Create a Warehouse
- Go to **Inventory > Warehouses**.
- Click **New Warehouse**.
- Enter "Main Warehouse" and save.

## 2. Create a Category
- Go to **Inventory > Categories**.
- Click **New Category**.
- Enter "Electronics" and save.

## 3. Create a Product
- Go to **Inventory > Products**.
- Click **New Product**.
- Name: "Bluetooth Speaker"
- SKU: "BS-01"
- Reorder Level: "10"
- Save. 

*(At this point, you have 0 stock. If you check the Dashboard, the AI CEO will issue a Critical warning!)*

## 4. Add Initial Stock
- Go to **Inventory > Adjustments**.
- Click **New Adjustment**.
- Select Type: **RECEIPT**
- Select "Bluetooth Speaker" and "Main Warehouse".
- Quantity: "50".
- Save.

## 5. Check the AI CEO
- Go to the **Dashboard**.
- The AI CEO will now report that your stock levels are healthy!
